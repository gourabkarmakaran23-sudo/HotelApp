Want to develop a Hotel & Restrurant Software(Professional and full package) using Lattest DotNet Core and Angular Framework and want to
used PostgresSQL or SQLServer as a database server(I think PostgresSQL is better) 

1. Design the Database first using EntityFrameworkCore code first approach.
2. Used Solid Principle, Dry Principle, Repository Design Pattern, Singletone Design Pattern, UnitofWork framework.
3. Use ErrorHandlingMiddleware, Logger, FluentValidation, JWT Authentication
4. Used DotNet CLI to create project and add all the dependecies.
5. Use ItextSharp or some other free tool to export the report in PDF.
6. Used any to tool to export Excel or CSV for data replated report.
7. Give it Professional Look
