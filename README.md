# Hotel & Restaurant Software - Architecture Proposal

This workspace contains the initial architecture design for a professional Hotel & Restaurant management system built with .NET Core and Angular.

## What is included

- `Architecture.md`: Detailed architecture and solution structure
- Project roadmap for backend and frontend
- Recommended stack, design patterns, and database modeling approach

## Goals

- Use **Entity Framework Core Code First** for database design
- Apply **SOLID**, **DRY**, **Repository Pattern**, **UnitOfWork**, and **Singleton** design concepts
- Implement **global error handling middleware**, **JWT authentication**, **FluentValidation**, and **logging**
- Provide **PDF export** and **Excel/CSV export** capabilities
- Target **PostgreSQL** as the preferred database
- Build a **modern professional UI** in Angular

## Next step

Open `Architecture.md` and start by scaffolding the solution with the recommended .NET CLI commands. After the solution layout is created, begin implementing the domain model and persistence layer.
